The package contains three files, fresh_tomatoes.py, entertainment_center.py and media.py


The file entertainment_center.py holds a list of movies and attributes for class Movie.  This is the file that should be run. It imports all or part of the other two files and can be opened in Idle.


The file media contains all of the rules and requirements for class Movie.  This class holds the name, storyline, poster and youtube preview attributes as well as the function to play the youtube links.


The file fresh_tomatoes creates and launches the website for the final product.
